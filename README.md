Telco Customer Churn Prediction Pipeline

## 📌 Project Overview
In the telecommunication sector, it is much cheaper to maintain existing customers than to attract new ones. The project aims at developing an end-to-end Machine Learning pipeline to identify customer churn (whether or not a customer will remain or leave).

With the help of the Telco Customer Churn dataset, we clean the data, conduct exploratory data analysis (EDA), feature engineering, and model training to determine at-risk-customers. The ultimate answer gives viable business information to enhance retention strategies.

## 📂 Dataset Details
* **Name of Data set: Telco Customer Churn.
* **Origin: Kaggle / IBM Sample Data Sets.
* **Rows:** 7,043 customers
* **Columns: 21 features (Demographics, Services, Account Info).
* **Target Variable: Churn (Yes/No)

## 🛠️ Technologies & Libraries
It is written in Python in a Jupyter Notebook (Google Colab) environment.
* **Data Manipulation: panda, numpy.
* **Visualization:** `matplotlib`, `seaborn`
* **Machine Learning: scikit-learn (Random Forest, Logistic Regression)
